# -Hyperspectral-Image-Mosaicking
一个简单的算法复现仓库，[欢迎浏览博客](https://www.cnblogs.com/Goblinscholar)。

## 项目结构

| 模块 | 说明 |
|------|------|
| `harris/` | Harris 角点探测 |
| `SIFT/` | SIFT 特征检测与匹配 |
| `RANSAC/` | RANSAC 误匹配剔除基线 |
| `USAC/` | USAC 通用采样一致性算法 |
| `Stitch_v1/` | **图像拼接：SIFT → USAC → Stitching** |
